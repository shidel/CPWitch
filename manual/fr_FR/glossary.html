Glossaire : La Référence de Poche

Pour comprendre les prophéties de la Sorcière, il faut maîtriser son langage. Voici les termes qui hantent le plus souvent son chaudron :

Codepage (Page de code)
Une table de valeurs indiquant quel caractère est représenté par une séquence de bits. Indispensable pour communiquer avec les anciens esprits (systèmes hérités).
Scryer (Scruter)
Le processus par lequel la Sorcière fouille un répertoire pour trouver des fichiers cachés et les charger dans le chaudron.
Transmutation
La conversion magique d'un texte de l'Unicode universel vers une page de code locale spécifique (ou vice-versa).
Le Présage (Icônes d'état)
Signes visuels indiquant l'état d'un fichier : Vert (Prêt), Jaune (Avertissement/erreur de format) ou Rouge (Erreur critique).
Unicode (UTF-8)
La langue moderne universelle du texte, englobant presque tous les caractères de toutes les langues de la Terre.
