Codepage vers Unicode : Précognition

Bien que la Sorcière soit souvent utilisée pour créer des fichiers hérités, elle est tout aussi douée pour les sauver. Ce rituel consiste à prendre un vieux texte encodé en Codepage et à le "promouvoir" vers l'Unicode moderne (UTF-8), le rendant compatible avec les éditeurs de bureau et les standards web d'aujourd'hui.

1. Scrutin Statistique

Lorsqu'un fichier hérité est jeté dans le chaudron, il en ressort généralement avec une icône Bleue . Contrairement aux fichiers Unicode qui contiennent des signatures d'encodage claires, les fichiers hérités ne sont souvent que des octets bruts. Pour les identifier, la Sorcière effectue une analyse statistique du texte afin de "deviner" sa langue et sa configuration régionale (Locale) probables.

2. Graphiques Circulaires de Probabilité

Comme la Sorcière fait une supposition éclairée, le volet Enchantements est un peu différent pour les fichiers bleus. Vous remarquerez des Graphiques Circulaires Bleus/Rouges :

  • La part bleue : Indique la probabilité statistique que le fichier appartienne à cette page de codes spécifique en se basant sur des modèles linguistiques connus.
  • La part rouge : Indique la probabilité que la sélection soit une erreur.
  • Le Choix : Si la Sorcière est très confiante, elle présélectionnera le candidat le plus probable, mais vous devriez toujours consulter les miroirs pour vérifier.

3. Le Miroir de Promotion

Le Lecteur Unicode (le miroir supérieur) est votre outil le plus important lors de ce rituel. Il vous montre exactement à quoi ressemblera le texte une fois promu en UTF-8.

  • Si vous voyez du texte déformé ou des symboles incorrects dans le miroir supérieur, la Sorcière a peut-être sélectionné la mauvaise page de codes de départ.
  • Essayez de sélectionner différents enchantements dans la liste jusqu'à ce que le Lecteur Unicode reflète le texte tel qu'il était initialement prévu d'être lu.

4. Finaliser la Promotion

Une fois que le texte apparaît correctement dans le miroir supérieur, cliquez sur l'icône Exporter . La Sorcière enveloppera le texte dans un sceau UTF-8 moderne, garantissant qu'il puisse être lu par n'importe quelle application moderne tout en préservant les fins de ligne d'origine.

Note : Si le texte ne contient que des caractères 7 bits standard, l'icône deviendra Grise . Ces fichiers sont universels et ne nécessitent pas de scrutin complexe pour être promus.
