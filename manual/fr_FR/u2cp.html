Unicode vers Codepage : L'Art de la Transmutation

C'est le rituel le plus courant pratiqué par la Sorcière : prendre un texte moderne encodé en Unicode (UTF-8) et le transmuer dans une page de codes DOS spécifique. Ce processus garantit que vos traductions et documents s'affichent correctement sur du matériel ancien et des émulateurs DOS.

1. Le Scrutin Initial

Lorsque vous jetez pour la première fois un fichier dans le chaudron, il apparaît avec une icône Orange . Cela indique que la Sorcière analyse actuellement son contenu. Une fois qu'elle a confirmé que le fichier est en Unicode, l'icône se transformera en Vert .

Note : Si le fichier ne devient pas vert, la Sorcière a détecté une complication :

  • Jaune indique un Augure de formatage (comme l'absence d'un saut de ligne final) qui doit être corrigé avant de continuer.
  • Bleu ou Gris indique que le fichier est déjà un fichier de Codepage ou ASCII. Voir Codepage vers Unicode pour ces rituels.
  • Rouge indique un Vide ou une erreur. Reportez-vous aux Icônes d'État pour plus de détails.

2. Choisir l'Incantation

Le volet Enchantements se remplira d'une liste de pages de codes potentielles. La Sorcière les classe à l'aide de Graphiques Circulaires de Compatibilité pour une référence visuelle rapide :

  • La part verte : Représente la portion de votre texte qui peut être convertie avec succès vers cette page de codes.
  • La part rouge : Représente les caractères qui n'existent pas dans cette page de codes et qui seront perdus.
  • Le Sceau Parfait : Lorsqu'une page de codes offre une compatibilité à 100 %, le graphique circulaire est remplacé par une Coche .

3. Identifier l'Invisible

Si vous sélectionnez une page de codes avec moins de 100 % de compatibilité, regardez attentivement le Lecteur de Codepage (le miroir inférieur). Tout caractère que la page de codes sélectionnée ne peut pas comprendre sera rendu par ¿. Cela vous permet de traquer les caractères problématiques avant que le sceau ne soit finalisé.

4. Finaliser le Sceau

Une fois que vous êtes satisfait de l'aperçu, cliquez sur l'icône Exporter . La Sorcière préservera les fins de ligne trouvées dans le fichier original et vous demandera une destination. La transmutation est alors effectuée, laissant votre source Unicode originale intacte.
