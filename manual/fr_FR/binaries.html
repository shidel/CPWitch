La Forge : Téléchargement et Compilation

Pour la plupart des utilisateurs, les « Esprits » (binaires) pré-construits sont le moyen le plus simple d'invoquer la Sorcière. Pour les forgerons, voici les rituels.

1. Les Esprits Pré-compilés

Les binaires officiels (Windows, macOS, Linux) sont disponibles ici :

  • Télécharger : https://up.lod.bz/CPWitch

2. Les Outils de l'Alchimiste

Vous aurez besoin de Lazarus IDE (v4.0+) et de Free Pascal (FPC). Voir le README pour le MPLA Framework.

3. La Forge Primaire (macOS)

La version « Master » est forgée sur macOS avec des scripts automatisés :

  • Transmutation d'images : Optimisation de 500+ images.
  • Génération d'unités : Création de icons.pp et version.pp.

4. La Forge Universelle (Windows & Linux)

Une fois les ressources générées, le projet peut être compilé immédiatement sous Windows ou Linux.

5. Acquérir la Source

Le code source est sur GitHub :

  • Source : https://github.com/shidel/CPWitch
