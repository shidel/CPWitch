La Forge : Téléchargement et Compilation

La Codepage Witch est un artefact open-source. Vous pouvez soit télécharger une version déjà forgée, soit utiliser les matières premières (code source) pour la créer vous-même.

1. Artefacts terminés (Binaires)

Pour ceux qui souhaitent commencer la magie immédiatement, des versions précompilées sont disponibles sur la page des sorties GitHub. Nous proposons des paquets pour :

  • Windows : Un fichier .exe autonome (64 bits).
  • macOS : Un "App Bundle" pour les systèmes Intel et Apple Silicon.
  • Linux : Des binaires compilés pour les distributions courantes.

2. Conjurer la Sorcière (Compilation)

Si vous souhaitez modifier le code ou construire l'application pour une autre plateforme, vous aurez besoin des outils suivants :

  • Lazarus IDE : La version 4.0 ou supérieure est recommandée.
  • Free Pascal Compiler (FPC) : La colonne vertébrale de la logique sorcière.

Le Processus :

  1. Clonez le dépôt depuis GitHub.
  2. Ouvrez le fichier de projet (cpwitch.lpi) dans Lazarus.
  3. Sélectionnez votre système d'exploitation et l'architecture de votre processeur dans les paramètres du projet.
  4. Appuyez sur Compiler (F9) pour forger le binaire.
