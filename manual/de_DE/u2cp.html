Unicode zu Codepage: Die Kunst der Transmutation

Dies ist das am häufigsten von der Hexe durchgeführte Ritual: Die Umwandlung von modernem, Unicode-kodiertem Text (UTF-8) in eine spezifische DOS-Codepage. Dieser Prozess stellt sicher, dass Ihre Übersetzungen und Dokumente auf historischer Hardware und in DOS-Emulatoren korrekt angezeigt werden.

1. Das anfängliche Scrying

Wenn Sie zum ersten Mal eine Datei in den Kessel werfen, erscheint sie mit einem orangenen Symbol. Dies zeigt an, dass die Hexe gerade den Inhalt analysiert. Sobald sie bestätigt, dass die Datei Unicode ist, verwandelt sich das Symbol in ein grünes Symbol.

Hinweis: Wenn die Datei nicht grün wird, hat die Hexe eine Komplikation entdeckt:

  • Gelb weist auf ein Formatierungs-Omen hin (wie einen fehlenden abschließenden Zeilenumbruch), das vor dem Fortfahren behoben werden sollte.
  • Blau oder Grau zeigt an, dass die Datei bereits eine Codepage- oder ASCII-Datei ist. Siehe Codepage zu Unicode für diese Rituale.
  • Rot weist auf eine Leere oder einen Fehler hin. Weitere Einzelheiten finden Sie unter Status-Icons.

2. Wahl der Beschwörung

Das Verzauberungen-Fenster wird mit einer Liste potenzieller Codepages gefüllt. Die Hexe bewertet diese mithilfe von Kompatibilitäts-Kreisdiagrammen als schnelle visuelle Referenz:

  • Das grüne Segment: Repräsentiert den Teil Ihres Textes, der erfolgreich in diese Codepage konvertiert werden kann.
  • Das rote Segment: Repräsentiert Zeichen, die in dieser Codepage nicht existieren und verloren gehen werden.
  • Das perfekte Siegel: Wenn eine Codepage 100 % Kompatibilität bietet, wird das Diagramm durch ein Häkchen ersetzt.

3. Das Unsichtbare identifizieren

Wenn Sie eine Codepage mit weniger als 100 % Kompatibilität auswählen, schauen Sie genau in den Codepage-Betrachter (den unteren Spiegel). Jedes Zeichen, das die gewählte Codepage nicht versteht, wird als ¿ gerendert. Dies ermöglicht es Ihnen, problematische Zeichen aufzuspüren, bevor das Siegel finalisiert wird.

4. Das Siegel finalisieren

Sobald Sie mit der Vorschau zufrieden sind, klicken Sie auf das Exportieren Symbol. Die Hexe bewahrt die in der Originaldatei gefundenen Zeilenenden und fragt Sie nach einem Zielort. Die Transmutation wird dann durchgeführt, wobei Ihre ursprüngliche Unicode-Quelle unberührt bleibt.
