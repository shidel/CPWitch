Die Schmiede: Herunterladen und Kompilieren

Die Codepage Witch ist ein Open-Source-Artefakt. Sie können entweder eine fertig geschmiedete Version herunterladen oder die Rohstoffe (Quellcode) nutzen, um sie selbst zu erschaffen.

1. Fertige Artefakte (Binärdateien)

Für diejenigen, die sofort mit der Magie beginnen möchten, stehen vorkompilierte Versionen auf der GitHub Releases-Seite zur Verfügung. Wir bieten Pakete für:

  • Windows: Eine eigenständige .exe-Datei (64-Bit).
  • macOS: Ein App-Bundle für Intel- und Apple Silicon-Systeme.
  • Linux: Binärdateien, die für gängige Distributionen kompiliert wurden.

2. Die Hexe selbst beschwören (Kompilieren)

Wenn Sie den Quellcode modifizieren oder für eine andere Plattform bauen möchten, benötigen Sie die folgenden Werkzeuge:

  • Lazarus IDE: Version 4.0 oder neuer wird empfohlen.
  • Free Pascal Compiler (FPC): Das Rückgrat der hexischen Logik.

Der Prozess:

  1. Klonen Sie das Repository von GitHub.
  2. Öffnen Sie die Projektdatei (cpwitch.lpi) in Lazarus.
  3. Wählen Sie Ihr Ziel-Betriebssystem und Ihre CPU-Architektur in den Projekteinstellungen.
  4. Drücken Sie Kompilieren (F9), um die Binärdatei zu schmieden.
