O Grimório: Um Guia de Conjuração

Há já algum tempo que venho gerindo o Projeto de Tradução do FreeDOS (também conhecido como FD-NLS) no GitHub. É aqui que são geridas as traduções para os pacotes fornecidos para o FreeDOS™.

Os colaboradores do FD-NLS são fantásticos, mas por vezes submetem apenas traduções em ficheiros de texto Unicode (codificados em UTF-8). Quase todos os programas DOS utilizam ficheiros de texto ASCII codificados por páginas de códigos (Codepages). Converter certas línguas é um pesadelo porque os editores modernos muitas vezes não suportam a página de códigos necessária.

Para resolver isto, nasceu a Codepage Witch. Ela não é um programa DOS; é uma aplicação de desktop moderna para macOS, Linux e Windows.

Embora a sua função principal seja converter ficheiros, ela realiza muita magia nos bastidores, desde decifrar o idioma até recomendar a página de códigos correta para a conversão.

Embora a Codepage Witch não seja difícil de usar, ela é um pouco diferente e pode ser confusa ao início. Este é o seu Grimório, que visa ajudar os seus amigos a compreendê-la melhor.
