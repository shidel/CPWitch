Codepage para Unicode: Precognição

Embora a Bruxa seja frequentemente usada para criar ficheiros legados, ela é igualmente habilidosa em resgatá-los. Este ritual envolve pegar em texto antigo codificado em Codepage e "promovê-lo" para o moderno Unicode (UTF-8), tornando-o compatível com os editores de desktop e padrões web de hoje.

1. Escrutínio Estatístico

Quando um ficheiro legado é lançado no caldeirão, normalmente emerge com um ícone Azul . Ao contrário dos ficheiros Unicode que contêm assinaturas de codificação claras, os ficheiros legados são muitas vezes apenas bytes brutos. Para identificá-los, a Bruxa realiza uma análise estatística do texto para "adivinhar" o seu idioma e configuração regional (Locale) prováveis.

2. Gráficos de Tarte de Probabilidade

Como a Bruxa está a fazer uma suposição informada, o painel de Encantamentos parece um pouco diferente para ficheiros azuis. Notará Gráficos de Tarte Azuis/Vermelhos :

  • A Fatia Azul: Indica a probabilidade estatística de o ficheiro pertencer a essa página de códigos específica com base em padrões linguísticos conhecidos.
  • A Fatia Vermelha: Indica a probabilidade de a seleção ser um erro.
  • A Escolha: Se a Bruxa estiver muito confiante, pré-selecionará o candidato mais provável, mas deve sempre consultar os espelhos para verificar.

3. O Espelho de Promoção

O Visualizador Unicode (o espelho superior) é a sua ferramenta mais importante durante este ritual. Ele mostra-lhe exatamente como o texto ficará uma vez promovido para UTF-8.

  • Se vir texto distorcido ou símbolos incorretos no espelho superior, a Bruxa pode ter selecionado a página de códigos de origem errada.
  • Tente selecionar diferentes encantamentos na lista até que o Visualizador Unicode reflita o texto como foi originalmente pretendido ser lido.

4. Finalizando a Promoção

Assim que o texto aparecer corretamente no espelho superior, clique no ícone Exportar . A Bruxa envolverá o texto num selo UTF-8 moderno, garantindo que possa ser lido por qualquer aplicação moderna, preservando os finais de linha originais.

Nota: Se o texto contiver apenas caracteres de 7 bits padrão, o ícone ficará Cinzento . Estes ficheiros são universais e não requerem um escrutínio complexo para serem promovidos.
