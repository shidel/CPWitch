A Forja: Download e Compilação

Para a maioria dos utilizadores, os "Espíritos" (binários) pré-compilados são a forma mais fácil de invocar a Bruxa. No entanto, para aqueles que desejam entrar na forja e construir o programa a partir do código-fonte, alguns rituais devem ser observados.

1. Os Espíritos Pré-compilados

Binários oficiais para os "3 Grandes" sistemas operativos (Windows, macOS e Linux) podem ser encontrados no seguinte local:

  • Download: https://up.lod.bz/CPWitch

2. As Ferramentas do Alquimista

Para compilar o código-fonte, precisará do Lazarus IDE (Versão 4.0 ou superior) e do compilador Free Pascal (FPC). Informações sobre os frameworks necessários (como o MPLA Framework) e outras dependências podem ser encontradas no README do projeto.

3. A Forja Primária (macOS)

Atualmente, a versão "Mestre" do código é forjada em macOS. Este ambiente utiliza scripts automatizados que gerem as necessidades complexas de recursos do projeto e criam ficheiros de construção essenciais:

  • Transmutação de Imagens: O projeto contém aproximadamente 500 imagens. No macOS, estas são processadas, escaladas e otimizadas automaticamente para a interface.
  • Geração Dinâmica de Unidades: Os scripts de construção geram automaticamente unidades Pascal internas necessárias para a compilação, como icons.pp, version.pp e version.def.
  • O Grande Recurso: Estas imagens e dados de versão são compilados num ficheiro de Recursos do Lazarus (.lrs) massivo.

4. A Forja Universal (Windows & Linux)

Embora a geração inicial de recursos exija o ambiente macOS, o código-fonte resultante é verdadeiramente multiplataforma. Uma vez geradas essas unidades dinâmicas e ficheiros de recursos, o projeto pode ser aberto no Lazarus em Windows ou Linux e compilado imediatamente.

5. Adquirindo a Fonte

O código-fonte da Bruxa está alojado no GitHub:
