A Forja: Download e Compilação

A Codepage Witch é um artefacto de código aberto. Pode descarregar uma versão já forjada ou usar as matérias-primas (código-fonte) para a criar você mesmo.

1. Artefactos terminados (Binários)

Para aqueles que desejam começar a magia imediatamente, as versões pré-compiladas estão disponíveis na página de lançamentos do GitHub. Oferecemos pacotes para:

  • Windows: Um ficheiro .exe independente (64 bits).
  • macOS: Um pacote de aplicação para sistemas Intel e Apple Silicon.
  • Linux: Binários compilados para as distribuições mais comuns.

2. Conjurar a Bruxa (Compilação)

Se desejar modificar o código ou construí-lo para uma plataforma diferente, precisará das seguintes ferramentas:

  • Lazarus IDE: Recomenda-se a versão 4.0 ou superior.
  • Free Pascal Compiler (FPC): A espinha dorsal da lógica da bruxa.

O Processo:

  1. Clone o repositório do GitHub.
  2. Abra o ficheiro de projeto (cpwitch.lpi) no Lazarus.
  3. Selecione o seu sistema operativo e a arquitetura do CPU nas definições do projeto.
  4. Pressione Compilar (F9) para forjar o binário.
