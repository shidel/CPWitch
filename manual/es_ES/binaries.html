La Forja: Descarga y Compilación

Para la mayoría, los "Espíritus" (binarios) pre-construidos son la forma más fácil de invocar a la Bruja. Sin embargo, para aquellos que deseen entrar en la forja, deben observarse algunos rituales.

1. Los Espíritus Pre-compilados

Los binarios oficiales para los "3 Grandes" sistemas (Windows, macOS y Linux) se encuentran aquí:

  • Descarga: https://up.lod.bz/CPWitch

2. Herramientas del Alquimista

Necesitarás Lazarus IDE (v4.0+) y Free Pascal (FPC). Consulta el README del proyecto para detalles sobre el MPLA Framework.

3. La Forja Primaria (macOS)

La versión "Maestra" se forja en macOS usando scripts automatizados:

  • Transmutación de Imágenes: Procesa y optimiza más de 500 imágenes.
  • Generación Dinámica: Crea unidades Pascal esenciales como icons.pp y version.pp.

4. La Forja Universal (Windows y Linux)

Una vez generados los recursos, el código es multiplataforma y puede compilarse inmediatamente en Windows o Linux con Lazarus.

5. Adquisición de la Fuente

El código fuente reside en GitHub:

  • Fuente: https://github.com/shidel/CPWitch
