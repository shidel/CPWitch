La Forja: Descarga y Compilación

La Codepage Witch es un artefacto de código abierto. Puede descargar una versión ya forjada o tomar las materias primas (código fuente) para crearla usted mismo.

1. Artefactos terminados (Binarios)

Para aquellos que deseen comenzar con la magia de inmediato, las versiones precompiladas están disponibles en la página de lanzamientos de GitHub. Ofrecemos paquetes para:

  • Windows: Un archivo .exe independiente (64 bits).
  • macOS: Un paquete de aplicación para sistemas Intel y Apple Silicon.
  • Linux: Binarios compilados para las distribuciones más comunes.

2. Conjurar a la Bruja (Compilación)

Si desea modificar el código o construirlo para una plataforma diferente, necesitará las siguientes herramientas:

  • Lazarus IDE: Se recomienda la versión 4.0 o superior.
  • Free Pascal Compiler (FPC): La columna vertebral de la lógica de la bruja.

El Proceso:

  1. Clone el repositorio desde GitHub.
  2. Abra el archivo de proyecto (cpwitch.lpi) en Lazarus.
  3. Seleccione su sistema operativo y arquitectura de CPU en los ajustes del proyecto.
  4. Presione Compilar (F9) para forjar el binario.
