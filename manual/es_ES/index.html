El Grimorio: Una Guía de Conjuración

Durante bastante tiempo, he estado gestionando el Proyecto de Traducción de FreeDOS (también conocido como FD-NLS) en GitHub. Aquí es donde se gestionan las traducciones para los paquetes proporcionados para FreeDOS™.

Los colaboradores de traducción en FD-NLS son un grupo fantástico, pero a veces solo envían traducciones como archivos de texto Unicode (codificados en UTF-8). Casi todos los programas de DOS utilizan archivos de texto ASCII codificados por páginas de códigos (Codepages). Convertir algunos idiomas es una verdadera pesadilla porque los editores modernos a veces no admiten la página de códigos requerida.

Finalmente, tuve que hacer algo al respecto, y así nació la Codepage Witch. No es un programa de DOS; es una aplicación de escritorio moderna para macOS, Linux y Windows diseñada para resolver estos problemas y mucho más.

Si bien la función principal de la Codepage Witch es convertir archivos de texto, realiza mucha magia entre bastidores para facilitar el proceso, desde descifrar el idioma hasta recomendar la página de códigos adecuada.

Aunque la Codepage Witch no es muy difícil de usar, es un poco diferente y puede resultar confusa al principio. Este es su Grimorio, cuyo objetivo es ayudar a sus amigos a comprenderla mejor.
